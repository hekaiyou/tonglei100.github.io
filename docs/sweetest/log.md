# Sweetest 更新日志


## 版本 0.8.50 (2019.03.27)

- 支持在<>中对 list 切片取值

  > 如已通过自定义脚本赋值了 a = `[[1,2,3],[4,5,6]]`, 则 `<a[0][1]>` 值为 `2`

- 支持在<>中取 dict 的键值

  > 如已通过自定义脚本赋值了 apple = `{'color': 'red'}`, 则 `<apple['color']>` 值为 `'red'`

## 版本 0.8.41 (2019.03.01)

 详情见文档**全屏及元素截图比较**章节。
